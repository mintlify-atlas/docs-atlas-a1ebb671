# Boards API

The Boards API provides functionality for creating, managing, and collaborating on boards. Boards are the primary organizational containers for lists and cards in Better Todo.

## Queries

### getAll

Retrieves all boards accessible to the current authenticated user. Returns only active (non-archived) boards where the user is a member.

<ParamField query="handler" type="async function">
  No parameters required. Authentication is handled automatically.
</ParamField>

<ResponseField name="boards" type="array">
  Array of board objects with member roles
  <Expandable title="Board Object Properties">
    <ResponseField name="_id" type="Id<'boards'>" required>
      Unique board identifier
    </ResponseField>
    <ResponseField name="title" type="string" required>
      Board title
    </ResponseField>
    <ResponseField name="description" type="string | undefined">
      Board description
    </ResponseField>
    <ResponseField name="color" type="string | undefined">
      Board color (hex or named color)
    </ResponseField>
    <ResponseField name="visibility" type="'private' | 'team' | 'public'" required>
      Board visibility setting
    </ResponseField>
    <ResponseField name="createdBy" type="string" required>
      User ID of board creator
    </ResponseField>
    <ResponseField name="archived" type="boolean" required>
      Archive status (always false in this response)
    </ResponseField>
    <ResponseField name="role" type="'owner' | 'admin' | 'member' | 'viewer'" required>
      Current user's role on this board
    </ResponseField>
    <ResponseField name="createdAt" type="number" required>
      Unix timestamp of creation
    </ResponseField>
    <ResponseField name="updatedAt" type="number" required>
      Unix timestamp of last update
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```typescript Example Usage
import { useQuery } from "convex/react";
import { api } from "../convex/_generated/api";

function BoardsList() {
  const boards = useQuery(api.boards.getAll);
  
  return (
    <div>

          <h3>{board.title}</h3>
          <span>Role: {board.role}</span>
        </div>
      ))}
    </div>
  );
}
```

### getById

Retrieves a single board by ID with all its lists and cards. Includes card metadata such as checklist progress and label associations.

<ParamField query="boardId" type="Id<'boards'>" required>
  The unique identifier of the board to retrieve
</ParamField>

<ResponseField name="board" type="object | null">
  Board object with nested lists and cards, or null if not found or unauthorized
  <Expandable title="Board Object Properties">
    <ResponseField name="_id" type="Id<'boards'>" required>
      Unique board identifier
    </ResponseField>
    <ResponseField name="title" type="string" required>
      Board title
    </ResponseField>
    <ResponseField name="description" type="string | undefined">
      Board description
    </ResponseField>
    <ResponseField name="color" type="string | undefined">
      Board color
    </ResponseField>
    <ResponseField name="visibility" type="'private' | 'team' | 'public'" required>
      Board visibility setting
    </ResponseField>
    <ResponseField name="role" type="'owner' | 'admin' | 'member' | 'viewer'" required>
      Current user's role on this board
    </ResponseField>
    <ResponseField name="lists" type="array" required>
      Array of list objects with cards
      <Expandable title="List Object Properties">
        <ResponseField name="_id" type="Id<'lists'>" required>
          Unique list identifier
        </ResponseField>
        <ResponseField name="title" type="string" required>
          List title
        </ResponseField>
        <ResponseField name="position" type="number" required>
          List position (for ordering)
        </ResponseField>
        <ResponseField name="cards" type="array" required>
          Array of card objects in this list
          <Expandable title="Card Object Properties">
            <ResponseField name="_id" type="Id<'cards'>" required>
              Unique card identifier
            </ResponseField>
            <ResponseField name="title" type="string" required>
              Card title
            </ResponseField>
            <ResponseField name="position" type="number" required>
              Card position within list
            </ResponseField>
            <ResponseField name="labelIds" type="array" required>
              Array of label IDs associated with this card
            </ResponseField>
            <ResponseField name="checklistCount" type="number" required>
              Number of checklists on this card
            </ResponseField>
            <ResponseField name="checklistItemsCompleted" type="number" required>
              Number of completed checklist items
            </ResponseField>
            <ResponseField name="checklistItemsTotal" type="number" required>
              Total number of checklist items
            </ResponseField>
          </Expandable>
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```typescript Example Usage
import { useQuery } from "convex/react";
import { api } from "../convex/_generated/api";

function BoardView({ boardId }) {
  const board = useQuery(api.boards.getById, { boardId });
  
  if (!board) return <div>Board not found</div>;
  
  return (
    <div>
      <h1>{board.title}</h1>
      <p>Your role: {board.role}</p>
      {board.lists.map(list => (
        <div key={list._id}>
          <h2>{list.title}</h2>
          <div>
            {list.cards.map(card => (
              <div key={card._id}>
                <h3>{card.title}</h3>
                <span>{card.checklistItemsCompleted}/{card.checklistItemsTotal} tasks</span>
              </div>
            ))}
          </div>
        </div>
      ))}
    </div>
  );
}
```
</CodeGroup>

### getArchived

Retrieves archived boards and cards for the current user. Can optionally be scoped to a specific board.

<ParamField query="boardId" type="Id<'boards'>" optional>
  Optional board ID to filter archived cards. If omitted, returns archived content from all boards.
</ParamField>

<ResponseField name="result" type="object">
  Object containing archived boards and cards
  <Expandable title="Result Properties">
    <ResponseField name="boards" type="array" required>
      Array of archived board objects
    </ResponseField>
    <ResponseField name="cards" type="array" required>
      Array of archived card objects with board metadata
      <Expandable title="Card Properties">
        <ResponseField name="_id" type="Id<'cards'>" required>
          Card identifier
        </ResponseField>
        <ResponseField name="title" type="string" required>
          Card title
        </ResponseField>
        <ResponseField name="boardTitle" type="string" required>
          Title of the board this card belongs to
        </ResponseField>
        <ResponseField name="boardColor" type="string | undefined">
          Color of the parent board
        </ResponseField>
        <ResponseField name="archived" type="boolean" required>
          Archive status (always true)
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```typescript Example Usage
import { useQuery } from "convex/react";
import { api } from "../convex/_generated/api";

function ArchivedView() {
  const archived = useQuery(api.boards.getArchived, {});
  
  return (
    <div>
      <h2>Archived Boards</h2>
      {archived?.boards.map(board => (
        <div key={board._id}>{board.title}</div>
      ))}
      
      <h2>Archived Cards</h2>
      {archived?.cards.map(card => (
        <div key={card._id}>
          {card.title} - {card.boardTitle}
        </div>

    </div>

}
```

    </ResponseField>

    </ResponseField>

## Role Hierarchy

Board members can have one of four roles, each with different permissions:

<Tabs>
  <Tab title="Owner">
    - Full control over the board
    - Can delete the board
    - Can archive/restore the board
    - Can manage all members and change roles
    - Can update board settings
    - Cannot be removed by other members
    - Only one owner per board (the creator)
  </Tab>
  <Tab title="Admin">
    - Can update board settings
    - Can add and remove members (except owner)
    - Cannot change other admins' roles
    - Cannot archive or delete the board
    - Can be removed by the owner
  </Tab>
  <Tab title="Member">
    - Can create, edit, and delete cards and lists
    - Can view all board content
    - Cannot manage members
    - Cannot change board settings
    - Can be removed by admins or owner
  </Tab>
  <Tab title="Viewer">
    - Can view all board content
    - Cannot create, edit, or delete content
    - Cannot manage members
    - Cannot change board settings
    - Can be removed by admins or owner
  </Tab>
</Tabs>

## Visibility Settings

Boards support three visibility levels:

- **Private**: Only invited members can access the board
- **Team**: All team members can discover and request access to the board
- **Public**: Anyone with the link can view the board (requires member role to edit)

## Error Handling

All mutations may throw the following errors:

- `"Unauthorized"` - User is not authenticated
- `"Insufficient permissions"` - User lacks required role for the operation
- `"Board not found"` - Specified board does not exist
- `"Member not found"` - Specified member does not exist
- `"User is already a member"` - Attempted to add a user who is already a member
- `"Only owners can..."` - Operation restricted to board owners
- `"Cannot remove board owner"` - Attempted to remove the board owner
- `"Cannot change owner role"` - Attempted to change the owner's role

```typescript Error Handling Example
import { useMutation } from "convex/react";
import { api } from "../convex/_generated/api";

function BoardOperations({ boardId }) {
  const updateBoard = useMutation(api.boards.update);

      await updateBoard({

        title: "New Title"
      });
    } catch (error) {
      if (error.message === "Insufficient permissions") {
        alert("You don't have permission to update this board");
      } else if (error.message === "Unauthorized") {
        alert("Please sign in to continue");
      } else {
        alert("An error occurred: " + error.message);
      }
    }
  };
  
  return <button onClick={handleUpdate}>Update</button>;
}
```
