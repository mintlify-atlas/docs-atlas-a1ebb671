<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold dark:text-gray-100 text-gray-900 mb-4">Quick start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get Better Todo up and running in minutes</p>
  
  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install dependencies using Bun or your preferred package manager.
      
      ```bash
      git clone https://github.com/MrDXTR/BetterTodo.git
      cd BetterTodo
      bun install
      ```
    </Step>
    
    <Step title="Set up Convex backend">
      Run the setup script to create and connect your Convex project.
      
      ```bash
      bun run dev:setup
      ```
      
      <Note>
        This will create a new Convex project and generate the necessary environment variables in `packages/backend/.env.local`.
      </Note>
    </Step>
    
    <Step title="Configure environment variables">
      Set up your authentication and email service credentials via the Convex dashboard or CLI.
      
      ```bash
      npx convex env set SITE_URL "http://localhost:3001"
      npx convex env set GOOGLE_CLIENT_ID "your-google-client-id"
      npx convex env set GOOGLE_CLIENT_SECRET "your-google-client-secret"
      npx convex env set RESEND_API_KEY "re_your_resend_api_key"
      ```
      
      <Tip>
        For detailed configuration options, see the [environment configuration](/configuration/environment) guide.
      </Tip>
    </Step>
    
    <Step title="Start the development server">
      Launch both the web app and Convex backend in development mode.
      
      ```bash
      bun run dev
      ```
      
      Open [http://localhost:3001](http://localhost:3001) to view your app. Sign up to create your first account, then grant yourself admin privileges using the Convex dashboard.
    </Step>
  </Steps>
</div>
